## 🚀 Projects

- ✈️ **FlyCombi**  
  System for airline route analysis modeled using graph theory. It computes optimal paths between cities, evaluates airport importance, handles itinerary planning with constraints, and exports routes to KML. Built in Python using classical graph algorithms such as Dijkstra, BFS, and Kruskal.  
  🔗 https://github.com/Agusb04/FlyCombi

- 🛫 **Algueiza Flight System**  
  Real-time flight management system implemented in Go, based on custom data structures such as HashMap, Binary Search Tree (BST), and Heap. Supports efficient querying, time-range filtering, flight lookup by code, and priority-based operations under strict algorithmic complexity constraints.  
  🔗 https://github.com/Agusb04/Algueiza-Flight-System

- 🕹️ **Yet Another Battle City**  
  Recreation of the classic Battle City game built in Java and JavaFX using MVC architecture. Includes singleplayer and local co-op modes, enemy AI, power-ups, collision system, custom levels, decoupled sound system, and automated testing.

- 📦 **Pedilo-Despues**  
  Full-stack order management system with a web application, Flask API, and mobile app built in Kivy. Supports order tracking, ticket generation with unique IDs, real-time status updates, and QR-based promotions. Uses MySQL for persistent storage.  
  🔗 https://github.com/Agusb04/Pedilo-Despues
